## KEY FINDINGS

### 1. Sled Push = the keystone failure (cascade)
- +37s on the clock, but the REAL cost was the downstream cascade: gassed legs bled time on Pull (+43s), BBJ (+18s), Row (+8s). True cost ~1:30–2:00, not 37s.
- Mark broke **2×/lane** (normally 1×/lane). Buckling carpet caused lift-off.
- Training sled (2:35 on buckling turf) did NOT translate to race.
- **Mark's own fix (correct):** go LOWER on the push — hips down, low drive angle — to avoid lifting on buckling carpet. Must be fault-proof / unstoppable regardless of surface.
- This is the #1 priority: fixing it recovers pull, BBJ, row, and post-sled runs automatically.

### 2. Wall Balls = officiating + mental, NOT fitness
- 10:24 ≈ Seoul, BUT: judge activated the WRONG target height mid-set → ~20 missed targets through no fault of Mark's → had to argue with judge + head judge → credited 15 reps back.
- The dispute caused mental pain, insecurity, loss of focus that bled into the rest of the set.
- **Real physical WB was ~9:20–9:30** (≈1min lost to the drama). Training WB was strong (7×15 unbroken fresh, ~4:30; race strategy 6×15+1×10 @ ~7s rest).
- **WB is NOT the fitness limiter it was at Seoul.** Fix = race-craft (target check before start, don't argue mid-set, composure/reset) + clean rep standard (full depth + clear target height so no judge CAN no-rep) + compromised breathing. NOT relentless raw volume.

### 3. Lunges = the clean WIN (proof the method works)
- −51s. Trained specifically for 6 weeks → converted directly. Same specific approach now goes to sled/WB.

### 4. BBJ = jump distance, not engine
- Mark's insight: "in doubles I jumped FURTHER." Longer jumps = fewer reps = less time + less metabolic cost.
- Fix = jump distance + turnaround efficiency, NOT volume. Shin/ankle-impact gated (post-surgical L ankle).

### 5. Running = already good, was too conservative
- Ran 4:45 holding HR 155–158 — but **158 = LT2 (threshold)**, so he was already AT threshold and STILL had reserve (only 31% Z4). Run total −58s.
- BKK lever isn't "push HR higher" — it's holding 4:38 on the POST-SLED runs where the cascade cost time. Fix the sled → the runs hold.

### 6. Niggles much improved
- Left forearm + right wrist: issues since Seoul, **both much better now.** Should limit training far less going forward.
- Less fatigue + much less soreness after BOTH races vs Seoul = genuine strength gain.

### 7. Shoe lesson
- Singles raced in BRAND-NEW Adidas Dropset Elite, never run >2km before — risky in an A-race. Worked out (great sled grip, confirmed good in doubles), but not ideal day-1.
- **Lesson: Dropset Elite proven (excellent sled grip) — keep for racing, but break in any race shoe with ≥1 full-distance run first.**
